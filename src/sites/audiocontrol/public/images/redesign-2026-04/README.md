# Redesign — April 2026 — Before / After Screenshots

Permanent record of the audiocontrol.org brand redesign (April 2026).

brand-rebuild workflow.

## Capture settings

- Viewport: 1440 × 900
- `scale: 'css'`, `type: 'png'`
- "Before" screenshots pulled from https://audiocontrol.org/ while that
  site was still serving the pre-redesign build from `main`.
- "After" screenshots pulled from a local `python3 -m http.server` serving
  `dist/audiocontrol/` (production build, no Astro dev toolbar overlay).
